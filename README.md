# 💳 Alke Wallet

**Alke Wallet** es una aplicación web que simula el funcionamiento de una billetera digital, permitiendo realizar operaciones básicas de gestión de dinero electrónico como inicio de sesión, depósitos, transferencias y visualización de movimientos.

---

## ✨ Características

- 🔐 Sistema de autenticación con validación de credenciales  
- 💰 Depósitos rápidos con montos predefinidos  
- 📤 Transferencias a contactos guardados  
- 👥 Gestión de contactos para facilitar envíos  
- 📊 Historial de transacciones con filtros y ordenamiento  
- 💅 Diseño responsive  
- 🎨 Animaciones suaves  
- 💾 Persistencia de datos mediante sessionStorage  
- ⚡ Validaciones en tiempo real  

---

## 🛠️ Tecnologías

- HTML5  
- CSS3  
- JavaScript ES6+  
- Bootstrap 5.3.8  
- jQuery 3.7.1

## 🧠 Caso de Estudio

### 🎯 Desafío principal

El principal desafío del proyecto fue simular el funcionamiento de una billetera digital, implementando operaciones financieras básicas y gestionando la información del usuario de manera estructurada, manteniendo una experiencia de usuario clara y funcional.

---

### 💡 Solución propuesta

Se desarrolló una aplicación web utilizando HTML, CSS y JavaScript, organizando la lógica en distintos módulos para cada funcionalidad (login, depósitos, transferencias y transacciones). Se utilizó `sessionStorage` para simular la persistencia de datos y se implementaron validaciones para asegurar la correcta interacción del usuario.
