# Proyecto_AlkeWallet

# 💳 Alke Wallet

**Alke Wallet** El presente proyecto consiste en el desarrollo de una billetera digital web denominada Alke Wallet, cuyo objetivo es simular operaciones básicas de gestión de dinero electrónico, tales como inicio de sesión, depósitos, envíos de dinero y visualización de movimientos.

## ✨ Características

- 🔐 **Sistema de autenticación** con validación de credenciales
- 💰 **Depósitos rápidos** con montos predefinidos
- 📤 **Transferencias** a contactos guardados
- 👥 **Gestión de contactos** para facilitar envíos
- 📊 **Historial de transacciones** con filtros y ordenamiento
- 💅 **Diseño responsive** con tema rosado moderno
- 🎨 **Animaciones suaves** y transiciones elegantes
- 💾 **Persistencia de datos** mediante sessionStorage
- ⚡ **Validaciones en tiempo real** para mayor seguridad

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Estilos personalizados con variables CSS y animaciones
- **JavaScript ES6+** - Lógica de la aplicación
- **Bootstrap 5.3.8** - Framework CSS responsive
- **jQuery 3.7.1** - Manipulación del DOM y eventos

## 📖 Uso

### 1. Acceso al Sistema

1. Abre `index.html` en tu navegador
2. Haz clic en el botón "Comenzar"
3. Ingresa tus credenciales en la pantalla de login

### 2. Navegación

- **Menú Principal:** Visualiza tu saldo y accede a todas las funciones
- **Depositar:** Agrega fondos a tu cuenta
- **Enviar Dinero:** Transfiere a tus contactos
- **Movimientos:** Consulta tu historial completo
